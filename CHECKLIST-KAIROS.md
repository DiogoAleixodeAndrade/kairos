# Checklist Kairos

## Antes de usar em produção

- [ ] `.env` não pode estar versionado no GitHub.
- [ ] `.env.example` deve existir sem chaves reais.
- [ ] Login e cadastro funcionando.
- [ ] Fluxo demo passando pelo onboarding.
- [ ] Onboarding salvando peso inicial, peso atual e meta.
- [ ] Home atualizando alimentação, água, treino, sono, progresso e IA.
- [ ] Perfil mostrando XP, nível e conquistas.
- [ ] Botão de sair funcionando.
- [ ] Supabase com RLS ativo.
- [ ] Edge Function da IA configurada com `OPENAI_API_KEY`.
- [ ] Testar no mobile com Expo Go.
- [ ] Testar build web.
- [ ] Testar build Android/iOS futuramente com EAS.
